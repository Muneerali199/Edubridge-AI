#### 1. Auth Service (âœ“ Started)

- JWT token issuance and validation
- Refresh token mechanism
- Password reset flow
- Role-based access control (RBAC)

#### 2. User Service (ğŸ“ To Build)

- Teacher profile management

#### 3. Content Service (ğŸ“ To Build)

- Prompt template management

- Safety filters and validation

**Tech:** Spring WebClient, Redis, OpenAI SDK

#### 6. Analytics Service (ğŸ“ To Build)
- Student progress tracking
- Skill vector computation
- KPI calculations
- Report generation
- Dashboard data aggregation
- Cohort analysis

**Tech:** Spring Boot, PostgreSQL

#### 7. Payments Service (ğŸ“ To Build)
- Subscription management
- Payment processing (Stripe/Razorpay)
- Invoice generation

- Coupon codes

### Data Stores

#### PostgreSQL
- **Schemas:**
  - `auth` - Users, tokens, verification
  - `users` - Profiles, students, teachers
  - `content` - Courses, lessons, resources
  - `assessment` - Questions, quizzes, attempts
  - `analytics` - Progress, metrics
  - `payments` - Subscriptions, invoices

#### Redis
- Session management
- JWT blacklist
- AI response caching
- Rate limiting counters
- Real-time leaderboards (future)

#### MinIO (S3-compatible)
- **Buckets:**
  - `lessons` - Video lessons, PDFs
  - `assets` - Images, diagrams
  - `uploads` - Student submissions
